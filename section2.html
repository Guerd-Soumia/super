How It Works

  1. If you have a ready-made project and only need an investor, choose Ready Projects.
  2. If you have a non-complete project and need our help to solve your problem, choose Project Needs.
  3. For Ready Projects, we take 5% of the project value if we find an investor.
  4. For Project Needs, we take 10% of the project value for our assistance.
